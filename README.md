# Crad-Game

## 1. Introdução
Este projeto e um jogo de multiplayer UNO card game desenvolvido para Android usando Jetpack Composo.

## 2: Technologias usadas
- Kotlin
- Jatpack Compose
- Firebase Authentication
- Firebase Firestore
- Android Studio

## 3. Regras do jogo implementadas
- Skip
- Reverse
- -Draw Two (+2)
- Wild
- Wild Draw Four
- Seleção de cor para Wild Cards
